Отчёт технического плана по С++

За 2022-06-08 по 2022-06-20

За данный период изучался CMake и gcc.

Cmake

Практиковал в основном CMake на Windows + Visual Studio. CMake + Tests.
Плюсы:

  1. Довольно удобно делает CMake, что генерирует исполняемый файла и проекта, и тестов.
Минусы:
  1. Неудобно писать CMakeLists на винде, нежели на UNIX. Либо приходится настраивать винду, либо ставить wsl.
  2. Установка google test на visual studio дико неудобная, нежели на UNIX. На винде нужно копировать из документации 3 команды, которые пулят неизвестно куда либу, когда как на UNIX - писать одну команду по факту.

GCC

gcc -- набор компиляторов для различных языков программирования, разработанный в рамках проекта GNU. gcc -- для С, g++ -- для С++. Хотя gcc поддерживает компиляцию С++.

Не могу сказать, что длинный и трудночитаемые варнинги и ошибки при компиляции программы -- минус, посколкьу все настраивается флагами при запуске команд. Ничего совершенно нового, к сожалению, не узнал, поскольку ранее изучил работу общую компилятора для плюсов.

Обучение не логгировалось :)
