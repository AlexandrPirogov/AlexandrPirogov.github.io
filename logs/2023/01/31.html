За данный период изучался фреймворк агррегаций MongoDB. Данный фреймворк позволяет исполнять агрегирующие функции аналогично SQL. Основные функции/выражения таковы:

  1. $unwind -- для каждого элемента массива выдает новый документ
  2. $project -- отвечает за проекцию элементов документа
  3. $match -- выражение, которое устанавливает предикат для документов
  4. $sum -- суммирование по полю
  5. $avg -- среднее по полю
  6. $group -- разделяет документы на группы в соответствии с ключом группировки

Концепция исполнения агрегирующих запросов в MongoDB представлена в виде пайплайном -- каждая агрегирующая функция есть пайплайн. Запрос может состоять из несколькоих подобных пайплайнов. Результат каждого пайплайна передается в следующий пайплайн и так до последнего пайплайна.

Единственный минус, который имеется при работе с агрегирующими запросами -- синтаксис. Писать в терминале подобные запросы тяжело, легко допустить ошибку.
