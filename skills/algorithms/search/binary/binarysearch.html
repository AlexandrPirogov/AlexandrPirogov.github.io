Бинарный поиск

Бинарный поиск -- алгоритм типа "разделяй и властвуй". Позволяет осуществлять быстрый поиск в отсортированном массиве за O(lg n).

Суть его в следующем: чтобы найти элемент q в массиве S, мы сравниваем q с элементом, находящийся в середине массива S, тобишь S[n/2]. Если значение ключа q меньше, чем значение ключа S[n/2], то этот ключ должен находиться в левой половине массива, иначе -- в правой.

Реализация

                    
typedef size_t LL;

const LL capacity = 10000000000;

int binary_search(std::vector<int>& arr, int key, int low, int high)
{
    if(low > high)return -1;
    
    int middle = (low+high)/2;
    if(arr[middle] == key)return middle;

    arr[middle] > key ? binary_search(arr, key, low, middle-1) : binary_search(arr,key, middle+1, high);
}

Заметки

Благодаря двоичному поиску можно быстро находить количество последовательных элементо в отсортированном контейнере.
