АТД Список

Список -- абстрактный тип данных, который представляет последовательность элементов Elem определенного типа ElemType. Первый элемент списка Elem1 -- первый элемент списка (head). Elemn -- последний элемент списка (tail). В случае n = 0 Имеем пустой список.

Важное свойства списка заключает в том, что его элементы можно линейно упорядочить в соответствии в с их позицией в списке. Если у нас имеется элемент Elemk, то Elem k-1 предшествует за Elemk, а Elem k+1 -- следует за Elemk.
