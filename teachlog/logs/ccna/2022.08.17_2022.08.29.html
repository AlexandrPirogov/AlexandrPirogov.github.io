Отчёт технического плана по изучению компьютерных сетей

За 2022-08-17 по 2022-08-29

За данный период было изучено:

  1. Изучен подробнее сетевой уровень по учебнику "Компьютерные сети, нисходящий подход":
    1. Работа маршрутизатора. Как данные поступают, что обрабатывает их, куда отсылает, устройство маршрутизатора.
    2. Способы передачи данных: установление виртуального канала или отправление дейтаграммных сетей
  2. DNS, DHCP. DNS -- система, для перевода human-readable адреса в IP. Состоит из нескольких уровней. Использует UDP. DHCP -- система/инструмент, позволяющий конечным хостам изучать различные аспекты настроек сети (конфигурации, маску, gateway и т.д).
  3. NTP -- система для синхронизации времени в сети интернет. Несмотря, что все устройства имеют внутрение "часы", необходима синхронации. Синронизируются со специальными серверами.
  4. ACLs -- инструмент для фильтрации пакетов. Можно настроить на входящии и исходящие "сообщения". Существует также Extended ACLs -- расширение ACL, также можно указывать фильтровать по адресу источника и адресу назначения.
  5. Syslog, SNMP -- инструменты для конфигурации сети. Syslog -- инструмент логирования, SNMP -- инструмент для конфигурации сети. SSH, Telnet -- инструменты для удаленного доступа к консоли устройства (используется для настройки).
