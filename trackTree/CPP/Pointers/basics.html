Рассмотрим следующий пример:
#include

int main()
{
int a = 5;
std::cout << "&a = " << &a << '\n';
int *pointerA = &a;

std::cout << "pointerA = " << pointerA << '\n';
std::cout << "*pointerA = " << *pointerA << '\n';
std::cout << "&pointerA = " << &pointerA << '\n';

return 0;
};
Результат работы программы будет таким:

&a = 0x7fff12668abc
pointerA = 0x7fff12668abc
*pointerA = 5
&pointerA = 0x7fff12668ac0

Переменная а равная 5 имеет адрес 0x7fff12668abc. Знак & берёт адрес переменной в памяти. Также, указатель pointerA имеет значения равное адресу переменной a. Чтобы получить значения ячейки, на который указывает pointerA, достаточно поставить знак * перед pointerA. Как мы видимо, значение *pointerA = 5. Также у pointerA, как и у любой переменной, имеется свой адрес = 0x7fff12668abc. Также, через указатель возможно изменять значение переменной, на который тот указывает. Рассмотрим следующий пример:
#include

int main()
{
int a = 5;
std::cout << "&a = " << &a << '\n';
int *pointerA = &a;

std::cout << "pointerA = " << pointerA << '\n';
std::cout << "*pointerA = " << *pointerA << '\n';
std::cout << "&pointerA = " << &pointerA << '\n';

*pointerA = 10;

std::cout << "a = " << a << '\n';
std::cout << "pointerA = " << pointerA << '\n';
std::cout << "*pointerA = " << *pointerA << '\n';
std::cout << "&pointerA = " << &pointerA << '\n';

return 0;
};

Результат работы программы будет таким:
&a = 0x7ffef4485a3c
pointerA = 0x7ffef4485a3c
*pointerA = 5
&pointerA = 0x7ffef4485a40

a = 10
pointerA = 0x7ffef4485a3c
*pointerA = 10
&pointerA = 0x7ffef4485a40

Как мы видим, изначально, значение переменной a было равно 5. Далее, мы получаем значение ячейки через указатель *pointerA и присваиваем новое значение = 10. Заметьте, что ни адрес переменной a, ни адрес pointerA не изменился.
