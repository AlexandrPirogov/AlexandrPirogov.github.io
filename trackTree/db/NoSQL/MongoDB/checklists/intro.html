MongoDB

MongoDB -- база данных, хранящая данные в виде javascript объектов. Относится к нереляционным.

С MongoDB можно создавать несколько баз данных в которых, по аналогии с таблицами в реляционных СУБД, могут хранится коллекции, хранящие данные в виде javascript объектов.
